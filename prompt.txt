# FR-XP-001

## XP Reward Engine

### Description

The system shall award **Experience Points (XP)** when users complete quests, milestones, streaks, and achievements.

XP is the primary progression currency.

---

## Objective

Create strong dopamine loops and visible progression.

Users should feel:

> “I’m getting stronger.”

---

## XP Sources

| Action                 |          XP |
| ---------------------- | ----------: |
| Daily Quest            |    10–40 XP |
| Main Quest             |   50–150 XP |
| Weekly Challenge       |  100–300 XP |
| Boss Battle            | 300–1000 XP |
| Arc Completion         | 500–5000 XP |
| Streak Bonus           |    Variable |
| First Completion Bonus |      +25 XP |
| Perfect Day            |     +100 XP |

---

## XP Difficulty Formula

XP depends on:

### Formula

XP=(BaseXP \times DifficultyMultiplier \times StreakMultiplier \times ArcMultiplier) + BonusXP

---

### Difficulty Multiplier

| Difficulty | Multiplier |
| ---------- | ---------: |
| Easy       |         ×1 |
| Medium     |       ×1.5 |
| Hard       |         ×2 |
| Legendary  |         ×3 |

---

### Example

Quest:

> Deep Work Session

Base XP:
20

Difficulty:
Hard

Streak:
1.5x

Final:

20 × 2 × 1.5 = **60 XP**

---

### Acceptance Criteria

✅ XP granted instantly
✅ Animation triggered
✅ UI updated in real-time
✅ Server validation applied

---

### Edge Cases

Prevent XP farming.

Quest cannot award XP twice.

Store locally.

Sync later.

# Example Store

```ts
@Injectable()
export class QuestStore {

 quests = signal<Quest[]>([]);

 completed = computed(() =>
   this.quests().filter(q => q.done)
 );

 completeQuest(id: string){
   this.quests.update(quests =>
      quests.map(q =>
         q.id === id
         ? {...q, done: true}
         : q
      )
   );
 }
}
```
