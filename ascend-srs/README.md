# Ascend — Enter Arc Mode

## Software Requirements Specification (SRS)

**A Premium Real-Life RPG for Self-Improvement**

---

## Product Overview

**Ascend** is a game-first self-improvement platform designed to help users improve their real-life habits, productivity, discipline, fitness, learning, and personal growth through RPG-style progression systems.

Users don't track habits — they complete quests, level up stats, compete in leagues, and evolve their identity through **Arc Mode**.

---

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | Angular 20+ / Ionic / Capacitor / PWA |
| Backend | Spring Boot (Java 21) |
| Auth | Firebase Authentication |
| Database | Firestore + PostgreSQL (Hybrid) |
| Notifications | Firebase Cloud Messaging (FCM) |
| Realtime | Spring WebSocket |
| Cache | Redis |
| Analytics | Firebase Analytics + Custom |
